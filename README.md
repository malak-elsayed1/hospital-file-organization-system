

# Hospital File Organization System

A C++ file management system for doctors and patients records implementing low-level indexing and efficient space reclamation using Avail Lists and First-Fit search strategies.

## Features
* **Record Management**: CRUD operations for Doctors and Patients.
* **Advanced Indexing**: Primary and Secondary indexing for high-speed retrieval.
* **Space Optimization**: Manual management of record offsets and Avail Lists to reuse deleted space.
